# Handball Club — Site officiel

Site web du club de handball, développé avec React, Vite, TypeScript et shadcn/ui.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui (Radix UI)
- React Router
- React Query

## Démarrage

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
